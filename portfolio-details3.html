Career Climb SPA

"Career Climb" career counseling website is a platform dedicated to guiding individuals in making informed career decisions. It offers personalized career advice, resources, and tools to help users identify their strengths, interests, and goals.

Features and Characteristics of Career Climb SPA

  • Modern Web Development Stack: Built using ReactJS for dynamic and responsive user interfaces, styled with Tailwind CSS, and integrated with Google Firebase for secure authentication.
  • Single Page Application (SPA): Ensures seamless navigation with fast loading times and a smooth user experience.
  • Flexible Authentication Options: Users can register and log in using either email/password or their Google account for convenience.
  • Private Route Protection: Service details and commenting features are secured behind private routes, requiring users to log in to access them. Unauthorized users are redirected to the login page.
  • Personalized Dashboard: Logged-in users can add services to their dashboard and easily manage them from a dedicated dashboard page.
  • Interactive Feedback System: Logged-in users can provide comments or feedback on service detail pages, enhancing engagement and interaction.
  • Fully Responsive Design: The website is optimized for all screen sizes, ensuring an excellent user experience on mobile, tablet, and desktop devices.

Frontend Packages Used

  • firebase: For user authentication and storing user data.
  • react: Core JavaScript library for building user interfaces.
  • react-dom: For rendering React components into the DOM.
  • react-helmet-async: For managing the document head (e.g., titles and meta tags) dynamically.
  • react-icons: To use icons in the user interface.
  • react-router: For navigation and routing.
  • react-router-dom: For handling routing in a web environment.
  • sweetalert2: For displaying elegant and interactive alerts.
  • swiper: For creating swiping galleries and sliders.

Tech Stack

  • Frontend: ReactJS
  • Backend: Express
  • Authentication: Firebase
  • Database: MongoDB (for storing food items and user data)
